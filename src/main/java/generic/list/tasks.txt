Используя знания о Generics, Interfaces реализуйте свои собственные уникальные коллекции
SimpleList и SimpleUniqueList.
-
SimpleList динамический список основанный на массиве.
который расширяется в два раза при его заполнении. Используя знания о Iterator реализовать fail-fast поведение.
-
SimpleUniqueList уникальный динамический список который не допускает хранения дубликатов.
Расширяется в два раза при его заполнении. Используя знания о Iterator реализовать fail-fast поведение.

На оба класса напишите unit тесты.
